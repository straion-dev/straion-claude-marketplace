# Straion Plugin for Claude Code

Integrate straion's requirements system with Claude Code workflows. Automatically have requirements availabel when working with specs and implementing them.

[![License: ELv2](https://shields.fly.dev/badge/license-ELv2-green)](https://github.com/straion-dev/straion-claude-plugin/blob/main/LICENSE.txt)

## Quick Start

### Installation

#### For Users (Install from GitHub)

Install the straion plugin directly from GitHub:

```bash
/plugin install https://github.com/straion-dev/straion-claude-plugin.git
```
