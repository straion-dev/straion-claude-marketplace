# Terminology

| Term            | Definition                                                                                  |
| --------------- | ------------------------------------------------------------------------------------------- |
| **Spec**        | Specification document (PRD, ADR, RFC, or lightweight feature spec), or implementation plan |
| **Task**        | Single implementation unit with clear acceptance criteria                                   |
| **Rule**        | Project rule from Straion that code/specs must comply with                                  |
| **Validation**  | Checking compliance against rules                                                           |
| **Compliance**  | Meeting rule constraints (✓ Compliant, ⚠ Partial, ✗ Violation)                              |
